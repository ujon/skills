# 연구노트 상세 작성 가이드

## 법적 근거

- **국가연구개발혁신법** 제35조
- **국가연구개발혁신법 시행령** 제65조
- **국가연구개발사업 연구노트 지침** (과학기술정보통신부 고시)

### 핵심 의무 사항
- 과학기술 분야 연구개발과제에 참여하는 연구자와 연구개발기관의 장은 연구노트를 작성·관리해야 한다.
- 계획서에 연구원으로 등록된 모든 인원이 개별적으로 작성해야 한다.
- 연구개발기관은 연구노트를 30년간 보관·관리해야 한다.

## 연구노트의 4대 핵심 요건

### 1. 기록 시점의 증명
- 작성 날짜를 명확히 기재
- 전자연구노트의 경우 타임스탬프 자동 기록
- 서면 연구노트의 경우 지워지지 않는 필기구로 날짜 기재

### 2. 기록자의 확인
- 기록자(연구원)의 서명 필수
- 소속, 직위 기재 권장

### 3. 제3자(점검자) 확인
- 해당 연구에 직접 참여하지 않은 제3자의 서명
- 기록 내용이 해당 시점에 존재했음을 객관적으로 증명
- 점검자는 내용을 이해할 수 있는 동일/유사 분야 전문가 권장

### 4. 위·변조 방지
- 수정 시 원문을 삭제하지 않고 취소선 처리
- 수정 사유와 일자 기재
- 빈 공간에 사선 표시하여 추후 기입 방지

## 섹션별 작성 요령

### 연구 목적 / 배경
작성 요령:
- 해당 연구/실험을 수행하게 된 배경과 필요성을 2~3문단으로 기술
- R&D 계획서 상의 세부 과제 또는 마일스톤과의 연계성을 명시
- 이전 실험/연구에서 도출된 과제가 있으면 언급
- 구체적인 연구 가설이나 검증 목표를 명시

예시 문구:
"본 연구는 [과제명]의 2차년도 세부과제 '[세부과제명]'과 관련하여, 
[구체적 기술/현상]의 [특성/성능]을 규명하기 위해 수행되었다.
선행 연구(연구노트 No.XX, 20XX.XX.XX)에서 확인된 [문제점/과제]를 
해결하기 위해 [접근 방법]을 적용하였다."

### 연구 내용 및 방법
작성 요령:
- 제3자가 동일한 실험/연구를 재현할 수 있도록 구체적으로 기술
- 사용 장비는 모델명, 제조사, 주요 설정값까지 기재
- 시약/재료는 제조사, 순도/등급, lot번호(가능한 경우)까지 기재
- 실험 절차를 시간순 또는 단계순으로 번호를 매겨 기술
- 모호한 표현 지양: "적당량" → "50 mL", "잠시" → "10분간"

예시 구조:
```
[사용 장비 및 재료]
- 분석 장비: XX사 모델 YY (S/N: ZZZZ)
- 시약: AA (BB사, 순도 99.5%, Cat. No. CCCC)
- 소프트웨어: DD v3.2 (EE Inc.)

[실험 절차]
1) 시료 XX를 YY 조건에서 ZZ분간 전처리
2) AA 용액 50 mL에 시료를 투입하고 BB°C에서 CC시간 반응
3) 반응 후 DD 방법으로 생성물 분리
4) EE 장비를 이용하여 FF 조건에서 분석 수행
```

### 결과
작성 요령:
- 측정 데이터를 표 또는 목록 형태로 정리
- 성공 결과뿐 아니라 실패 결과도 반드시 기록
- 데이터의 단위를 명확히 표시
- 그래프나 이미지가 있는 경우 "첨부 [번호] 참조"로 연결
- 반복 실험의 경우 각 시행의 결과를 모두 기록
- 정성적 관찰 사항도 기록 (색 변화, 이상 현상 등)

예시 구조:
```
[실험 결과]
- 시료 A: 강도 45.2 MPa (목표값 40 MPa 이상 → 달성)
- 시료 B: 강도 38.7 MPa (목표값 미달)
- 시료 C: 측정 중 장비 오류 발생, 재측정 필요

[관찰 사항]
- 시료 B에서 반응 온도 상승 시(80°C 이상) 변색 현상 관찰
- 해당 현상은 선행 문헌에서 보고되지 않은 사항으로, 추가 분석 필요
```

### 고찰 및 결론
작성 요령:
- 실험 결과를 연구 목적과 연계하여 해석
- 예상과 일치하는 결과와 그렇지 않은 결과를 구분하여 분석
- 실패한 경우 원인 분석과 가능한 해결 방안 제시
- 향후 연구 방향이나 후속 실험 계획을 기술
- 이 결과가 전체 과제 진행에 미치는 영향 언급

예시 문구:
"시료 A의 결과는 목표 성능을 달성하여, [공정/방법]의 유효성이 확인되었다.
반면, 시료 B는 목표에 미달하였으며, 이는 [원인 추정]에 기인한 것으로 판단된다.
향후 [조건 변경]을 통해 개선 여부를 확인할 예정이다."

### 특이사항 / 비고
기록 대상:
- 장비 고장, 캘리브레이션 이슈
- 실험 환경 변화 (온도, 습도, 정전 등)
- 연구 계획 대비 변경 사항과 변경 사유
- 외부 기관 의뢰 분석 내역
- 안전 관련 사고나 유의 사항

## 과제 평가와의 연관성

### 과정평가 지표
2022년부터 R&D 최종 평가에 성과와 과정평가를 동시 진행한다. 
과정평가 중 '연구 수행 방법 및 과정의 적절성' 지표에서 연구노트가 핵심 증빙이 된다.

### 성실실패 인정 요건
과제 수행 중 목표 미달성 시에도, 연구노트가 성실하게 작성되었다면 
'성실 실패'로 인정받아 과제비 환수 및 참여 제한 등의 제재를 면할 수 있다.

### 불성실 수행 판정 위험
연구노트가 부실하거나 없는 경우:
- 불성실 수행 판정
- 정부지원금 환수
- 향후 과제 참여 제한

## 파일명 규칙

생성하는 .docx 파일명은 다음 규칙을 따른다:
- 기본: `연구노트_[YYYY-MM-DD]_[순번3자리].docx`
- 과제명 포함 시: `연구노트_[과제약칭]_[YYYY-MM-DD].docx`
- 사용자가 별도 파일명을 지정하면 그것을 따른다
